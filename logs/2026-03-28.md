# 2026-03-28

## Observed issues
- Bumper elasticity set to 1.2 (>1) can inject energy and make the ball speed up unrealistically.
- Space has no damping; combined with high restitution can lead to energy blow-up over time.
- No headless physics regression test to catch NaNs/velocity explosions.

## Changes
- Added a small global damping factor to reduce runaway energy.
- Reduced bumper elasticity to a sub-1 value via settings constant.
- Added a headless unittest that simulates 10s and checks for finite positions/velocities and bounded speed.

## Parameter diffs
- `BUMPER_ELASTICITY`: (new) 0.95 (was hardcoded 1.2)
- `SPACE_DAMPING`: (new) 0.995

## Test results
